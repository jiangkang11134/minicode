## 四、结论

### 想理解项目，必须掌握这些类

```
第一层 core（必须懂）：
  ToolResult → ToolDefinition → ToolRegistry → AgentStep → ChatMessage → ModelAdapter → Store/AppState

第二层 product（重点）：
  SessionData → FileCheckpoint → MemoryManager → PermissionManager → ContextManager

第三层 decision（补充）：
  TurnRecurrentState → TurnStepPolicy → AssistantTurnDecision → ToolTurnDecision
```

### 按模块优先级

| 优先级 | 模块 | 原因 |
|:-----:|------|------|
| 🔴 必读 | `tooling.py` | 43 个模块引用，工具系统绝对核心 |
| 🔴 必读 | `types.py` | 10 个模块引用，核心类型定义 |
| 🔴 必读 | `turn_kernel.py` + `state.py` | agent 决策引擎 + 全局状态 |
| 🟡 重要 | `session.py` + `memory.py` | 产品特性（checkpoint/rewind/记忆） |
| 🔵 了解 | `permissions.py` + `context_manager.py` | 辅助系统 |
